O que é HTML?

HTML, ou Linguagem de Marcação de Hipertexto (HyperText Markup Language), é a linguagem fundamental utilizada na construção de páginas da web. Desenvolvida por Tim Berners-Lee na década de 1990, o HTML permite a criação e estruturação de documentos que podem ser interpretados por navegadores, definindo como o conteúdo deve ser apresentado ao usuário. Os documentos HTML são salvos com as extensões .html ou .htm e são fundamentais para a navegação na internet, pois permitem a criação de hiperlinks que conectam diferentes páginas e sites. A versão mais recente, HTML5, introduziu novas funcionalidades, como suporte nativo para áudio e vídeo, além de novas tags semânticas que melhoram a estruturação do conteúdo e a otimização para motores de busca (SEO).

TAGS

As tags HTML são os elementos fundamentais que definem a estrutura de uma página web. Cada tag HTML é composta por um nome de tag, que é cercado por colchetes angulares. Por exemplo, a tag <p> é usada para definir um parágrafo.

Tags podem ser de abertura e fechamento, como <h1>Título</h1>, ou auto-fechadas, como <br /> para uma quebra de linha.

TAGS de Texto

As tags de texto são usadas para formatar e estruturar o conteúdo textual de uma página HTML. Aqui estão algumas das principais tags de texto:

Tag de Parágrafo

A tag <p> é usada para definir um parágrafo. Exemplo: 

<p>Este é um parágrafo de texto.</p>

Tag de Cabeçalho

As tags de cabeçalho, de <h1> a <h6>, são usadas para definir títulos e subtítulos. Exemplo: 

<h1>Título Principal</h1>
<h2>Subtítulo</h2>

Tag de Negrito e Itálico

A tag <strong> é usada para texto em negrito, enquanto <em> é usada para texto em itálico. Exemplos:

Texto em Negrito

Texto em Itálico

Tag de Citação

A tag <blockquote> é usada para citações em bloco. Exemplo:

Esta é uma citação.

Atributos de TAGS

Os atributos são informações adicionais que podem ser adicionadas às tags HTML para modificar seu comportamento ou aparência. Eles são sempre especificados dentro da tag de abertura e seguem a sintaxe nome="valor". Por exemplo: 

<a href="https://www.exemplo.com">Clique aqui</a>

Os atributos podem ser obrigatórios ou opcionais. Um exemplo de atributo obrigatório é o href em uma tag de link, que especifica o destino do link.

Listas Ordenadas e Não Ordenadas

HTML permite a criação de listas ordenadas e não ordenadas. As listas ordenadas são criadas usando a tag <ol>, enquanto as listas não ordenadas usam a tag <ul>.

Lista Ordenada

  1. Item 1
  2. Item 2

Lista Não Ordenada

  • Item A
  • Item B

Links

Os links são criados usando a tag <a>. O atributo href é utilizado para especificar o destino do link. Por exemplo:

Visite o primeiro site da web!

Os links podem ser internos, levando a outras partes da mesma página, como na navegação no início desta página, ou externos, levando a outras páginas na web.
